How the Third Party Ecosystem Works

djangopackages.org

Different levels of support

  • A whole separate project e.g. Django Rest Framework
  • Done by a group e.g. django-debug-toolbar
  • By an individual e.g. django-mysql

Sometimes Merged

  • South -> Django Migrations
  • django-security -> Django Security Middleware

What about other Open Source projets?

Ruby on Rails

  • Similar, but no formal organization
  • All © David Heinemeier Hanson
  • Core team - but no documentation of how they are onboarded
  • No (obvious) funding

Ansible

  • Company behind it - sell consulting
  • Bought by Red Hat
  • All plugins in one bundle

MySQL

  • Originally separate, bought by Oracle
  • Open source license but not culture
  • Sell a premium version and consulting

MariaDB

  • Fork of MySQL
  • Open source license and culture
  • Sell consulting
